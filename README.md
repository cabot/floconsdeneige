# floconsdeneige
phpBB extension to make snow fall on the board.

## Minimum Requirements
* phpBB ≥ 3.3.x

## Install (easy)
1. Download the latest ready-to-install version [cabot_floconsdeneige.zip](https://github.com/cabot/floconsdeneige/releases/latest/download/cabot_floconsdeneige.zip).
2. Unzip the downloaded archive and upload the folder it contains into the `ext/` directory of your phpBB board.
3. Navigate in the ACP to `Customise -> Manage extensions`.
4. Look for `Flocons de neige` under the Disabled Extensions list, and click its `Enable` link.

## Install (advanced)
1. Download the latest release `Source code`.
2. Unzip the downloaded release, and change the name of the folder to `floconsdeneige`.
3. In the `ext/` directory of your phpBB board, create a new directory named `cabot` (if it does not already exist).
4. Copy the `floconsdeneige` folder to `/ext/cabot/`.
5. Navigate in the ACP to `Customise -> Manage extensions`.
6. Look for `Flocons de neige` under the Disabled Extensions list, and click its `Enable` link.

## Usage
1. Navigate in the UCP to `User Control Panel -> Board preferences -> Edit display options -> Display snowflakes -> Yes/No`.

## Uninstall
1. Navigate in the ACP to `Customise -> Extension Management -> Extensions`.
2. Look for `Flocons de neige` under the Enabled Extensions list, and click its `Disable` link.
3. To permanently uninstall, click `Delete Data` and then delete the `/ext/cabot/floconsdeneige` folder.
